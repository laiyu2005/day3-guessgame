| 概念   | 关键字/示例                              | 说明           |
| ---- | ----------------------------------- | ------------ |
| 条件语句 | `if / else if / else`               | 没有括号         |
| 循环   | `for {}` / `for i := 0; i < n; i++` | 唯一循环结构       |
| 随机数  | `math/rand` + `time.Seed`           | 伪随机，必须播种     |
| 无限循环 | `for { ... }`                       | 用 `break` 退出 |
| 输入数字 | `fmt.Scan(&guess)`                  | 指针传值         |
